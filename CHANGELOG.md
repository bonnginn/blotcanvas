# BlotCanvas バージョン履歴

このファイルには、公開版における主な変更を記録します。

## 0.2.8-beta.1 — 2026-07-23

### English UI

- Translated the full expanded “Data handling” notice
- Clarified local image processing, anonymous Cloudflare page-view analytics, download behavior, and browser compatibility in English

## 0.2.7-beta.1 — 2026-07-23

### 繰り返し条件

- 新規テンプレートの既定ペアを個別実験由来の `d / L` から一般的な `− / ＋` へ変更
- none、WTなどのグループラベルを、対応する2レーンの正確な中央へ配置
- グループ線を各グループ幅の内側へ短縮し、隣接する線が接続して見える問題を修正
- Figureプレビュー、SVG、編集可能PDF、画像書き出しへ共通適用

## 0.2.6-beta.1 — 2026-07-23

### English beta

- Added an English application at `?lang=en` while retaining the Japanese interface
- Added an English user guide and English README
- Updated the feedback form with bilingual Japanese/English titles, questions, choices, and data-sharing guidance

### 長い条件ラベル

- レーン幅と文字幅を比較し、1行・2段・90度回転を自動選択
- 2段表示では空白や記号など意味の切れ目に近い位置を優先して改行
- 長いラベルに応じて条件欄の高さを自動確保し、隣接ラベルとの衝突を回避
- Figureプレビュー、SVG、編集可能PDF、画像書き出しへ共通適用

## 0.2.5-beta.1 — 2026-07-23

### 出力サイズ

- 編集可能PDFを用紙一杯ではなく、後でFigureへ組み込むパネル実寸で出力
- 配置幅を25.5 mm、39 mm、54 mmから選択可能（従来案の約30%）
- グラフPNGにも選択した配置幅の物理解像度情報を付与し、背景を透過

### 領域指定

- 複数ゲル・複数抗体では「抗体1のGel 1 → Gel 2 → 抗体2のGel 1 → Gel 2」の順で未作成領域を追加
- Gel 2・抗体1が飛ばされ、横幅・高さの同期対象がずれる問題を修正
- 各領域で撮影セットの抗体候補をプルダウン選択可能にし、任意名称の直接入力も維持

## 0.2.4-beta.1 — 2026-07-23

### グラフ出力

- 選択中の抗体について、複数ゲルの測定済みグラフを一括出力
- Gel 1・Gel 2をゲル名付きで縦に並べた1枚の高解像度PNGとして保存
- 従来の現在グラフのみのPNG保存も継続

## 0.2.3-beta.1 — 2026-07-23

### PDF出力の修正

- 編集可能PDFで抗体名の文字幅をHelveticaの実際の字幅から計算
- GFP・GAPDHなどの右揃えラベルがブロットへ重なる問題を修正

## 0.2.2-beta.1 — 2026-07-23

### 公開版の修正

- 公開ビルドで非表示になっていたStep 8「定量化」とStep 9「グラフ」を有効化
- 開発版と公開版で定量化ワークスペースの入口が一致するよう修正

## 0.2.1-beta.1 — 2026-07-21

### 取り込み・マニュアル

- Merge非対応装置向けに、シグナル画像と明視野／マーカー画像から作業用Mergeを生成
- 作業用Mergeの生成時も元画像と定量用シグナル画像を保持
- 実験内容を含まない一般例と模式図を用いた日本語の文章マニュアルを追加
- 完成Figure例の条件、8レーン、分子量表示を整合
- アプリ上部へ「はじめての方へ」を追加

## 0.2.0-beta.1 — 2026-07-21

### 定量化（第1版）

- 複数ブロットの矩形ROI測定、共通背景補正、ゲル別の正規化
- 16-bit元画像からの測定と、補正済み作業プレビュー
- n.d.判定、飽和画素率、CSV出力、定量設定のJSON保存
- 条件表を利用した棒・ドットグラフ、階層ラベル、固定色パレット、PNG出力
- 大きな値の縦軸を指数表記へ変更

### Figure作成・操作性

- 自己完結SVGとIllustrator向け編集可能PDFを改善
- 高解像度PNG・JPEG・TIFF・PDF完成画像を追加
- Source recordをシグナル画像、Merge画像、または両方から選択可能
- 条件行の上下移動を追加
- Figureプレビューを見ながら条件・レーン・まとめラベル・抗体名を編集可能

### ベータ版としての位置づけ

- 実画像・複数ブラウザでの検証がまだ十分ではありません
- 現在の定量法は矩形ROIと共通背景を用いる初期実装です
- レーンプロファイル、baseline指定、ピーク積分など、定量方法の改善を予定しています
- 操作性と書き出し互換性には、引き続き改善の余地があります

## 0.1.3-beta.1 — 2026-07-20

### 条件表・利用統計

- 繰り返し条件の上位ラベルを対応レーンの中央へ配置
- d/Lなどのペア上の横線を短縮し、グループ間の間隔を改善
- レーンラベル位置設定を条件表画面へ常時表示
- 繰り返しや階層のない「各レーンに個別ラベル」方式を追加
- Cookieを使わないCloudflare Web Analyticsによる匿名ページビュー統計を導入
- 画像、プロジェクト内容、入力条件は利用統計へ送信しない旨を明記

## 0.1.2-beta.1 — 2026-07-20

### 編集可能な書き出し

- Illustrator向け編集可能PDFを追加
- ブロット画像をPDF内部に埋め込み、外部リンクを不要化
- 抗体名、条件ラベル、分子量、線、外枠を編集可能なPDF要素として保持
- 複数語のラベルを1つの文字オブジェクトとして扱えるよう調整
- SVGは互換性検証用として残し、編集可能PDFを推奨経路に変更
- 定量化機能は引き続き公開版には含めず、開発版でのみ利用可能

## 0.1.1-beta.1 — 2026-07-20

定量化を含まないFigure preparation公開ベータの更新版です。

### 改善

- Figureを高解像度PNG・JPEG・TIFF・PDFとして1ファイル出力
- 完成画像とFigureプレビューを同一のCanvas描画から生成し、ラベル位置の差を解消
- SVGを完成画像生成から切り離し、画像形式の書き出しを安定化
- SVGは自己完結型1ファイルとし、Illustrator互換性を試験中と明記
- Source recordをCLのみ、Mergeのみ、または両方から選択可能
- Source recordへ表示専用の自動階調補正を適用
- 条件ラベルと階層線の配置を改善
- 新規マーカープリセット登録欄を折りたたみ表示へ変更

### 未公開機能

- 開発中の定量化機能はこの公開版には含めていません。

## 0.1.0-beta.1 — 2026-07-20

最初の公開ベータ版です。

### 主な機能

- ATTO LuminoGraphのCL・Merge・M画像の自動分類
- その他の撮影装置に対する画像役割の手動割り当て
- TIFF・PNG・JPEG画像の読み込み
- ゲルごとの水平補正と、ブロット領域の切り出し
- Figure用画像の階調・明るさ・コントラスト調整
- 分子量マーカーの手動配置、プリセット選択、自作パターン登録
- 複数ゲル・複数抗体のFigure構成
- 階層、反復、±条件に対応したレーン条件表
- Gel 1から他のゲルへの条件表コピー
- Figure SVG、証拠保全SVG、作業プロジェクトJSONの保存
- ブラウザ内処理とPWA対応

### 今後の主要予定

- 元画像を使用したWestern blot定量化
- 定量値と解析記録のCSV出力
