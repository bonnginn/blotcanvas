BlotCanvas Beta — Getting started

Prepare editable Western blot figures, source records, and preliminary quantification locally in your browser.

Before you start

  • Keep an independent copy of every original acquisition file.
  • Use the latest Chrome or Edge. Safari support is being tested.
  • Save a project JSON periodically so the workflow can be resumed.
  • Figure display adjustments and quantitative measurements are separate. Quantification uses the original signal image.

BlotCanvas is beta software. Validate measurements against ImageJ or another established workflow before using results in a publication.

Development and feedback

BlotCanvas is created and improved with OpenAI Codex as a development tool. The developer reviews reports submitted through the Google form and may provide the report text to Codex to help organize the problem, investigate possible causes, and prepare implementation options.

A submitted request is not implemented automatically. The developer decides whether and when to make a change after considering its necessity, scientific validity, priority, and effects on existing workflows.

Do not enter or attach original Western blot images, unpublished data, personal or patient information, or other confidential information in the feedback form.

1. Import acquisition files

Choose the imaging system, then select a folder or individual image files. ATTO LuminoGraph files are grouped from the CL, Merge, and M naming pattern. For other systems, assign the signal, Merge, and marker/bright-field roles manually.

If the instrument does not create a Merge image, BlotCanvas can generate a working Merge from a signal image and a bright-field/marker image of identical pixel dimensions. The original files remain unchanged.

2. Correct rotation

For each image set and gel, click one marker band to place a horizontal guide. Adjust the angle until the marker band is parallel to the guide. Rotation remains independent for each gel.

3. Define blot regions

Create one region for each gel and antibody. Width is synchronized among regions assigned to the same gel; height is synchronized among regions assigned to the same antibody. Region position remains independent.

With multiple gels and antibodies, new regions are proposed in this order: Gel 1/antibody 1, Gel 2/antibody 1, Gel 1/antibody 2, Gel 2/antibody 2. Both gel and antibody assignments can be changed.

4. Adjust the Figure display

Invert the image and adjust brightness or contrast for presentation. Use the same settings for images probed with the same antibody when they are intended for direct comparison.

These controls affect Figure appearance only. Quantitative measurements use the unadjusted original signal pixels.

5. Place molecular-weight markers

Select the marker product and the molecular weights visible on each membrane. Click the corresponding bands on the full marker or Merge image from top to bottom. Marker presets can be registered and are stored in the browser.

6. Create lane labels

Set the lane count and choose a simple, repeated, hierarchical, or +/− template. Every generated label remains editable. Long labels are automatically kept on one line, wrapped onto two lines, or rotated by 90° according to available lane width.

7. Arrange and export

  • Editable PDF: embedded blot images with editable text, lines, and frames for Illustrator.
  • Editable SVG: a self-contained alternative whose Illustrator compatibility may vary.
  • PNG, TIFF, JPEG, PDF: self-contained final images.
  • Source record: original signal and/or Merge images with crop regions.
  • Project JSON: restores the working project later.

Select a physical placement width of 25.5, 39, or 54 mm for the editable PDF and graph outputs.

8. Preliminary quantification

Select a gel and blot, place one rectangular ROI over each lane’s band, and place a background ROI over an area without bands or artifacts. A loading control from the same gel can be selected for normalization.

The current implementation is an initial rectangular-ROI method. Lane profiles, baseline selection, and peak integration are planned.

9. Graph and save results

Lane labels provide the graph categories. Adjust the graph type, title, font, size, and fixed color palette, then export a transparent PNG or measurement CSV. Results for the same antibody across multiple gels can be combined into one PNG.

Statistical analysis and aggregation of independent biological replicates are not yet implemented.
